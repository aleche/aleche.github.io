基于Hexo的github.io页面的搭建步骤

发表于 2015-10-25   |   分类于 Programming   |   热度 °C

配置Git及GitHub环境

  • 注册一个GitHub账户，并建立一个名为user_name.github.io的repository

  • 配置Git

    • 终端下设置用户名密码

    • 生成密钥，需要指定路径

    • 命令执行成功后，在指定路径下会生成id_rsa和id_rsa.pub两个文件，用编辑器打开id_rsa.pub,拷贝内容添加GitHub账户中的Add SSH key

安装Hexo

  • 安装Hexo之前，需要先安装好Node.js

  • 安装Hexo，可参考官方文档

  • 在目标目录下初始化Hexo

  • 生成静态页面

  • 本地启动

在浏览器就能在http://localhost:4000看见生成的静态页面

部署在自己的github.io

  • 在目标目录下的站点配置文件_config.yml的deploy域中设置好自己的github.io信息

  • 生成部署

在浏览器中打开 http://user_name.github.io/ 验证效果
